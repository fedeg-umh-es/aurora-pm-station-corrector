# Scientific question

## Project
Aurora -> station PM local correction.

## Core question
Can a train-only local corrector improve PM10/PM2.5 peak skill relative to raw Aurora-AQ and persistence at urban Mediterranean stations?

## What this project is
- A reproducible scaffold for a new empirical project.
- A station-level evaluation and correction workflow.
- A first operational test of Aurora-AQ usefulness at local monitoring sites.

## What this project is not
- Not a novelty claim about rolling-origin itself.
- Not a novelty claim about H* itself.
- Not a Skill_VP project in this first phase.
- Not an Aurora fine-tuning project yet.

## Phase v0
- Smoke test for Aurora-AQ point extraction.
- Comparison against persistence and a lag-only local baseline.
- No datasets committed yet.
- No regional fine-tuning yet.

## Immediate decision rule
This repo only advances if raw Aurora-AQ can be aligned to station-level observations and evaluated reproducibly under train-only temporal validation.
